Getting Started

1 Backend Setup

bash 
cd backend
python3 -m venv venv
source venv/bin/activate
pip install fastapi uvicorn pydantic python-multipart --upgrade
uvicorn app.main:app --reload --port 8000

Compatible with Python 3.11+. Backend runs on localhost:8000

2 Frontend Setup

bash 
cd frontend
npm install
npm run dev

Next.js dashboard runs on localhost:3000
